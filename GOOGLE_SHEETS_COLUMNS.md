# Google Sheets Column Headers Reference

Copy and paste these headers into Row 1 of your Google Sheets.

## Sheet 1: Client Requirements

Copy this into Row 1:

```
Timestamp	Company Name	Contact Person	Email	Phone	City	Manpower Type	Number of Resources	Duration	Additional Details
```

Or as a comma-separated list:
```
Timestamp,Company Name,Contact Person,Email,Phone,City,Manpower Type,Number of Resources,Duration,Additional Details
```

---

## Sheet 2: Candidate Registrations

Copy this into Row 1:

```
Timestamp	Full Name	Phone	Email	Current City	Primary Skill	Years of Experience
```

Or as a comma-separated list:
```
Timestamp,Full Name,Phone,Email,Current City,Primary Skill,Years of Experience
```

---

**Note:** Make sure the headers match exactly (including capitalization and spacing) for the scripts to work correctly.
